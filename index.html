Brandstore

Brandstore is an Android application for Mall navigation

View on GitHub Download .zip Download .tar.gz

Introduction.

Brandstore is a One-stop-shop for Mall related activites. Primary features include :

  1. Catalog of all the Stores in a mall
  2. Details of type of products available in each store
  3. Take me there - Text-based Indoor mall navigation]

This project is the source code for the Brandstore Android application. You can find the app on Google Play

The Nodejs server code is also available here
